## 功能介绍（完整见运行截图）
管理员：基本功能包括登录、注册、退出，提供用户管理、剧本管理、订单管理、房间管理等模块，允许创建、编辑和删除用户及剧本信息，管理订单并生成统计报告以辅助决策。用户：能够登录、注册和退出系统，访问个人中心查看和修改个人信息，浏览和参与剧本活动，管理预约信息，查看历史订单和处理中的订单。
